The World Wide Web

The World Wide Web (WWW) is an information space where web resources (such as images, documents, websites, etc) are identified by Uniform Resource Locators (URLs) and made accessible via the Internet.

Some facts about the World Wide Web:

  • Invented by Tim Berners-Lee in 1989
  • First web browser created in 1990 at CERN
  • First web browser was released to the general public August 1991
